# Code of Conduct

## Our Pledge

We, as contributors and maintainers of **Niffler**, pledge to make participation in this project a respectful, harassment-free experience for everyone, regardless of background, experience level, or identity.

Niffler is an open-source project built on collaboration, patience, and shared curiosity. We expect all participants to uphold those values.

---

## Expected Behavior

All contributors, users, and maintainers are expected to:

- Be respectful and considerate in speech and actions
- Provide constructive feedback
- Accept differing viewpoints and experiences
- Focus on what is best for the project
- Assume good intent, especially during technical disagreement
- Be patient — this project is experimental and evolving

---

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment, discrimination, or personal attacks
- Dismissive or hostile language
- Trolling, flame wars, or baiting
- Publishing private information without consent
- Repeatedly ignoring project guidelines or maintainers’ direction

This applies to all project spaces, including:
- GitHub issues
- Pull requests
- Discussions
- Any related community spaces

---

## Scope

This Code of Conduct applies within all project spaces and when representing the project publicly.

---

## Enforcement

Project maintainers are responsible for clarifying standards and enforcing this Code of Conduct fairly and consistently.

Actions may include:
- Warning
- Temporary restriction
- Permanent removal from the project

Maintainers’ decisions are final.

---

## Reporting Issues

If you experience or witness behavior that violates this Code of Conduct, please report it privately to the project maintainers.

Reports will be handled discreetly and respectfully.

---

## Attribution

This Code of Conduct is inspired by the Contributor Covenant but adapted specifically for the Niffler project.

---

## Final Note

Disagreement is welcome.  
Disrespect is not.

We’re here to build something meaningful — together.
